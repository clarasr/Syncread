## How to replace the placeholders
1. Open any placeholder `.tsx` component: it renders a `<div>` mentioning the file path. Replace the component body with the
   original JSX and exports.
2. Placeholder `.ts` modules export a function called `<name>Placeholder` that throws. Replace the file with the real logic
   and delete the placeholder export.
3. Update `client/index.html`, CSS, and server files with the actual implementations from your local copy.
4. Re-upload the binary assets listed above.

Once you have restored everything locally, run `git status` to review the changes, then commit and push.
